# Word Generator

這是一個使用 Swift 開發的單詞生成器應用程式。

## 功能特點

- 生成隨機單詞
- 自定義單詞生成規則
- 用戶友好的界面

## 系統需求

- iOS 15.0 或更高版本
- Xcode 13.0 或更高版本
- Swift 5.0 或更高版本

## 安裝說明

1. 克隆此專案到本地：

```bash
git clone [repository-url]
```

2. 使用 Xcode 打開 `wordGenerator.xcodeproj` 文件

3. 選擇目標設備或模擬器

4. 點擊運行按鈕或使用快捷鍵 `Cmd + R` 來運行應用程式

## 專案結構

```
wordGenerator/

│   ├── Models/
│   ├── Views/
│   └── ViewModels/
```
